# vue-cli-app

## Project setup
```
npm install
```

### First, start json server
```
npm run jserver
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
